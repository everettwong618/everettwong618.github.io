## Regenerate
```bash
python assets/img/_make_og.py
```

## Metadata
`index.html` sets `og:title`, `og:description`, `og:image`, `og:image:width` (1200), `og:image:height` (630), and `twitter:card` = `summary_large_image`.

After deploy, refresh LinkedIn’s Post Inspector / Facebook Sharing Debugger if the old preview is cached.
